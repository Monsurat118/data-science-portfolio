Data Science Portfolio – Monsurat Adejumo

Welcome to my Data Science & Machine Learning portfolio.
This repository contains selected academic projects completed during my Machine Learning diploma program at NorQuest College.
Each project demonstrates knowledge of Python, data cleaning, machine learning, visualization, and problem-solving.

Projects Included
1️ Go Auto Used Car Price Prediction

Goal: Predict the sale price of a used car using features like mileage, year, make, and model.
Skills: Regression modeling, EDA, feature engineering, sklearn.

2️ NPRI Environmental Release Prediction

Goal: Predict pollutant release quantities using the Canadian NPRI dataset.
Skills: EDA, data transformation, feature selection, machine learning modeling.

3️ NSWO Migration Analysis (Owl Movement Data)

Goal: Analyze migration patterns using telemetry data from Northern Saw-whet Owls.
Skills: Time-series analysis, clustering, ML modeling, geospatial visualization.

4️ LEGO Image Analysis

Goal: Use image processing techniques to segment and analyze LEGO pieces in an image.
Skills: OpenCV, thresholding, segmentation, image preprocessing.
